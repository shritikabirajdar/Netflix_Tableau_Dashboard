# Netflix Dashboard in Tableau

## Project Overview
This project presents an interactive Netflix Dashboard created using Tableau to analyze movies and TV shows available on Netflix.

The dashboard provides insights into:
- Movies vs TV Shows distribution
- Ratings analysis
- Top genres
- Country-wise content distribution
- Yearly content growth trends
- Content details and filtering

---

## Tools & Technologies
- Tableau Public / Tableau Desktop
- CSV Dataset
- Data Visualization
- Dashboard Design

---

## Dataset
Dataset used:
Netflix Titles Dataset from Kaggle

Columns used:
- Title
- Type
- Rating
- Release Year
- Date Added
- Genre
- Country
- Duration

### KPI & Filters

## Key Insights
- Movies contribute more content than TV Shows on Netflix.
- TV-MA is the most common rating category.
- Documentaries and Stand-Up Comedy are among the top genres.
- Content addition increased rapidly after 2016.
- USA has the highest amount of Netflix content.
